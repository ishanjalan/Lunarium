Eemie's Classy New Interior (Custom Furniture Edition) - Version 1.0.0
An official conversion of Eemie's Classy New Interior furniture for CF, converted by Amariel.

:: Required Mods::
SMAPI (https://smapi.io/)
Custom Furniture (https://www.nexusmods.com/stardewvalley/mods/1254), which requries:
PyTK (https://www.nexusmods.com/stardewvalley/mods/1726)

:: What This Mod Does::
-Adds new furniture items via Custom Furniture, purchasable from Robin or catalogue

::Note::
-Currently 2 items are not coded properly, two small coffee tables. This is because Customfurniture does not currently allow for items where the rotations have differing sizes across tiles to function correctly.  This may be fixed in a CF update.

:: Installation Instructions::
1. Download and install the latest version of SMAPI
2. Download and install the latest version of PyTK
3. Download and install the latest version of Custom Furniture
4. Copy '[CF] Eemie's Classy New Furniture' folder to Mods
5. You're done! Run the game.
