Eemie's Flowers for Json Assets - Version 1.2.0
Adds Eemie's flowers to the game as new craftables.

:: Required Mods::
SMAPI (https://smapi.io/)
Json Assets (https://www.nexusmods.com/stardewvalley/mods/1720)

:: Installation Instructions::
1. Download and install the latest version of SMAPI
2. Download and install the latest version of Json Assets
3. Copy '[JA] Eemie's Flowers' folder to Mods
4. You're done! Run the game.
