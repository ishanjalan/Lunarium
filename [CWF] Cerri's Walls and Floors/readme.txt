Cerri's Walls and Floors (Custom Walls and Floors Edition) - Version 1.0.0
A conversion of Cerri's Walls and Floors for CWF.

:: Required Mods::
SMAPI (https://smapi.io/)
Custom Walls and Floors (https://www.nexusmods.com/stardewvalley/mods/3010)

:: Installation Instructions::
1. Download and install the latest version of SMAPI
2. Download and install the latest versions of PyTK & Custom Walls and Floors
3. Copy '[CWF] Cerri's Walls and Floors' folder to Mods
4. You're done! Run the game.
