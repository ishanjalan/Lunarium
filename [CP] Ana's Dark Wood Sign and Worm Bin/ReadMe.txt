[CP] Ana's Dark Wood Sign and Worm Bin - 1.0.0

Dark wood recolors of the Wood Sign and Worm Bin.
Pairs well with Eemie's Craftables and Mi's Victorian Furniture.

FEATURES:
- Dark wood reskin of the yellow toned Wood Sign
- Dark wood reskin of the green worm bin

REQUIREMENTS:
- SMAPI (https://smapi.io)
- Content Patcher (https://www.nexusmods.com/stardewvalley/mods/1915)

TO INSTALL:
- Install latest version of SMAPI
- Install Content Patcher
- Download and unzip into Stardew Valley/Mods
- Run the game with SMAPI

UNINSTALL:
